Angular Grid - Getting Started

Overview

Тhe Grid component for Angular is a powerful widget that displays tabular data. It offers rich support for interacting with data, including paging, grouping, sorting, filtering and editing.

Installation

I. First, install jQWidgets dependency:

npm install jqwidgets-scripts --save

II. By now we have a fully working Angular CLI app. But we need to modify it a bit so we can use the jQWidgets Angular Grid. Open the angular.json file and inside the styles property adds this line: 

"node_modules/jqwidgets-scripts/jqwidgets/styles/jqx.base.css"

It should look like this: 

    "styles": [
        "src/styles.css",
        "node_modules/jqwidgets-scripts/jqwidgets/styles/jqx.base.css"
    ],

III. Now open the tsconfig.json file and add these properties at the bottom (after the "compilerOptions"): 

    "include": [
        "src/**/*"
    ],
    "files": [
        "src/app/app.module.ts",
        "node_modules/jqwidgets-scripts/jqwidgets-ts/angular_jqxgrid.ts"
    ]

IV. Now we are ready to use jqxGrid in Angular application:
