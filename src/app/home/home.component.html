jQWidgets Angular Grid

Outlook-Style Data Grouping

Our Data Grid widget enables users to group data with simple drag and drop. Users can also rearrange the grouping hierarchy by dragging and dropping the grouping headers in the group panel.

Nested Data Grids

jqxGrid has a very flexible layout that enables easy nesting of any content, UI elements and widgets including nested data grids.

Row Details

Row details are built-in feature of the Angular data grid. You can change the height of the row details area and use custom data templates. You can also use jqxGrid APIs to expand and collapse the grid row details area.

Column Types

The infrastructure of the Angular Grid is designed to support multiple column types. The data grid provides built-in 'Text', 'Number', 'DropDownList', 'DateTime' and 'CheckBox' columns. Our Angular grid also supports custom data rendering. Developers can easily replace the default rendering logic and use custom columns and cells rendering.

Pinned Columns

jqxGrid supports Excel-like pinned (frozen) columns. This feature also allows you to pin columns to the left the gridview's canvas. The pinned columns are very useful when working with large tables and it makes it easy to scroll and compare data from different parts of the table.

Cells Formatting

Our Angular grid offers a cell formatting feature where you can easily specify a formatting string for grid column. You can format dates, numeric values like decimal numbers, integer and floating-point numbers, currency and percentage values.

Localization

Our Angular grid supports localization for multi-language web sites and applications. All text elements used in control buttons, menu items and elements can be localized.
